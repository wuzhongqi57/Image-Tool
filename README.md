# Image Tool

通用图像查看与预处理工具。支持大量图片的懒加载浏览、单图/多图算法处理、流水线预设、批量导出、成对裁剪。

## 安装

```bash
pip install flask pillow numpy opencv-python
```

或使用 conda：

```bash
conda create -n fusion_tool python=3.10
conda activate fusion_tool
pip install flask pillow numpy opencv-python
```

## 启动

```bash
python app.py
```

或双击 `run.bat`（Windows，自动激活 conda 环境并打开浏览器）。

默认地址：<http://127.0.0.1:5000>

## 使用

### 加载图片

- **路径加载**：在顶部输入框粘贴文件夹路径，点击"加载"
- **拖拽加载**：拖拽文件夹或图片文件到窗口
- **浏览加载**：点击 📁 按钮，通过目录浏览器选择文件夹

支持 PNG / BMP / JPG。首次加载仅扫描文件列表（不读取图片内容），点击缩略图时才按需加载，适合大规模数据集浏览。可同时加载多个文件夹，各自独立的缩略图条。

### 模式切换

顶部有两个模式标签：

| 模式 | 用途 |
|------|------|
| 🔬 模拟 | 算法实验、参数调试、流水线设计、结果预览 |
| 🏭 生产 | 批量处理、成对裁剪（数据集制备） |

### 模拟模式

1. 在左侧算法面板下拉菜单中选择算法
2. 点击 `?` 查看算法说明和参数含义
3. 调整参数滑块
4. 点击 ▶ 运行，结果出现在主视图

**流水线**：点击"＋ 加入流水线"将当前算法和参数加入管线，可串联多个算法。管线步骤可点击编辑或删除。支持保存/加载为预设。

### 生产模式

**批量处理**：选择预设 → 指定源文件夹和导出目录 → 点击"▶ 批量处理"。将预设中的流水线应用到源文件夹的全部图片，按原名导出。

**成对裁剪**：加载两个图片一一对应的文件夹（如 HQ + LQ 数据集），工具自动检测配对。在 HQ 视图上拖拽裁剪框，LQ 按 scale 同步裁剪。用于制备成对训练数据。

### 保存

- 模拟模式：点击 💾 保存当前视图为 PNG
- 生产模式：成对裁剪自动保存 HQ + LQ 到指定目录

## 内置算法

### 单图算法

| 算法 | 说明 | 典型用途 |
|------|------|---------|
| USM 锐化 | 非锐化掩模，经典锐化 | 超分后处理、纹理增强 |
| CLAHE | 自适应直方图均衡 | 红外对比度增强 |
| 双边滤波 | 保边平滑去噪 | 红外去噪保留热源边缘 |
| Gamma 校正 | 非线性亮度映射 | 暗区提亮、过曝修正 |
| 亮度/对比度 | 线性调整 | 快速亮度修正 |
| 高斯模糊 | 高斯核平滑 | 预处理降噪 |

### 多图算法

| 算法 | 说明 | 输入 |
|------|------|:--:|
| Alpha 混合 | 逐像素加权平均 | 2 图 |
| 加权混合 | 自定义权重混合 | 2+ 图 |
| 频域分离融合 | 低频/高频分离后分别混合 | 2 图 |
| 边缘引导融合 | 边缘区和平坦区用不同权重 | 2 图 |
| 金字塔融合 | 多尺度拉普拉斯金字塔融合 | 2 图 |

## 预设

预设保存完整的流水线配置（算法链 + 参数）。预设文件存储在 `presets/` 目录下。

- 保存：调整好流水线后，输入预设名称，点击"📌 保存预设"
- 加载：从下拉菜单选择已保存的预设
- 预置预设：`presets/default_presets.json`

## 配置

编辑 `config.py` 修改默认路径和参数：

```python
OUTPUT_DIR = "./output"      # 保存目录
PRESETS_DIR = "./presets"    # 预设目录
CACHE_SIZE = 4               # 图片缓存数量
HOST = "127.0.0.1"
PORT = 5000
```
